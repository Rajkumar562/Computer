Tags Description
< p > Creates a paragraph of text of base font size 100%.
< pre > Creates a block of preformatted text that preserves white space and is displayed in a monospaced font.
< i > Displays the content in italics.
< b > Displays the content in bold.
< strong > Defines the text with strong importance. Content is displayed in bold.
< mark > Defines the text as highlighted.
< small > Defines smaller text. Makes text one font size smaller in the HTML Document.
< sub > Defines the content as a subscript that appears half a character below the normal line.
< sup > Defines the content as a superscript that appears half a character above normal line.
< br > An empty element that starts a new line of text. Inserts a single line break.
